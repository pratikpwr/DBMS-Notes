# Basic Queries

- Lists all the databases available  
  `SHOW DATABASES;`

- Create a new database  
  `CREATE DATABASE <database_name>;`

- Delete the database  
  `DROP DATABASE <database_name>;`

- Choose the database to use  
  `USE <database_name>;`

- To know which database is in use  
  `SELECT DATABASE();`

- To know warnings  
  `SHOW WARNIGS;`
